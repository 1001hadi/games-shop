# The GamesShop API

## This server build to serve data from mongoDB to the project front-End.

## Schemas:

- Users:

  -name

  -email

  -pwd

  -admin

- Games:

  -name

  -category

  -price

  -numOfPlayer

  -desc

  -qty

- Cart:

  -items

  -totalPrice

  -userID

## Routes

- usersRoutes:

  - CRUD User

- GamesRoutes:

  -get all
  -get specific
  -update by admin
  -delete by admin

- cartRoutes:

  -items
